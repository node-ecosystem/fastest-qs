# fastest-qs

Object version inspired to custom `fast-querystring` used by [Elysia](https://github.com/elysiajs/elysia) that it's inspired to original [fast-querystring](https://github.com/anonrig/fast-querystring).

## Install
```sh
yarn add fastest-qs
```
